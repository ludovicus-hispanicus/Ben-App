# Splash slideshow images

Drop tablet photos here as `01.jpg`, `02.jpg`, ... and uncomment the matching
`<div class="slide">` entries in [../../loading.html](../../loading.html).

Each slide can carry a `data-id` (museum number) and `data-rights` (copyright
holder) — both are shown right-aligned above the log panel while that slide
is visible.

Image guidance:
- 1600×900 minimum (window maximizes to display size)
- JPG, ~150–300 KB each (these go into every install)
- High-contrast tablets work best — the splash dims them by ~55%
- Avoid copyrighted images you don't have rights to redistribute
